## About

> デザイン専門学校でゲームクリエイターを目指して学習していたが、ある時 Web を少し学習してみると面白さの余り運命を感じた気がした、そこから私のフロントエンドエンジニアを目指す道が始まったのだ。

これまで Web は目覚ましい発展を遂げてきた、私が生まれ幾年経てばインターネットを用いたコンテンツがたちまち出現し、私の周りに溢れ始めた。 

私は誰かのクリエイティブな計画に挑戦し、誰かの理想を叶え、またそれを仕事とする人に対して強い尊敬の念を持っている。 

尊敬を抱く相手、その姿こそが私の理想と捉え、その位置に自らも立つためエンジニアを今日も目指していると考えている ELELINE であった。
